Questi template sono customizzazioni dei template xpt di metamodeldoc.
Per utilizzarli occorre sostituirli manualmente nel file del plugin:
net.randomice.metamodeldoc.generator_1.0.12.jar

Le customizzazioni sono state necessarie per elimiare la riga bianca in testa ai file html prodotti 
che generavano un errore se l'html era fornito dall'help di eclipse, ma potrebbero essere ulteriormente
customizzate per altre esigenze.
